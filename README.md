# ML Study

## Getting started

Create a python virtual env

```shell
python -m venv venv
```

Activate the virtual env

Windows:

```shell
# In cmd.exe
venv\Scripts\activate.bat
# In PowerShell
venv\Scripts\Activate.ps1
```

Linux and MacOS:

```shell
source venv/bin/activate
```

Install python libraries

```shell
pip install -r requirements.txt
```
